Currency Converter Web Application

A modern and responsive Currency Converter Web Application built using HTML, CSS, and JavaScript. This project allows users to convert currencies in real time using live exchange rates fetched from the Frankfurter API.

🚀 Features
🌍 Real-time currency conversion
💱 Supports multiple international currencies
🔄 Live exchange rate updates using API
📱 Responsive and user-friendly interface
⚡ Fast and lightweight web application
🎨 Clean modern UI design
🔢 Accurate conversion calculations


🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsive design
JavaScript – Functionality and API integration
Frankfurter API – Live currency exchange rates
